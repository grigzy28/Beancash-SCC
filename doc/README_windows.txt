Intro
-----

Bean Cash is a free open source project derived from Bitcoin, with
the goal of providing a long-term energy-efficient digital currency.
Built on the foundation of Bitcoin, PeerCoin, Novacoin, and Blackcoin - Bean Cash
furthers previous innovations and has pioneered the use of Static Block rewards
with Proof of Bean (The first v3.0 Proof of Stake cryptocurrency). Bean Cash stays
close to the original Bitcoin reference platform while continuing to provide 
inovations in th block chain space, while maintining block chain integrity.

More information can be found at:  http://www.beancash.org
